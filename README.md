# Zoo Puzzle Hub 🐾

A vibrant, educational logic game portal designed for kids, featuring a cute animal theme, engaging sound effects, rich animations, and built-in AI solvers!

[Русское описание ниже](#ru)

---

## 🎮 Included Games
- **Zoo 2048 🐾**: Match tiles, slide cute animals, and try to reach the Unicorn! (3x3 and 4x4 grid support).
- **Zoo 15-Puzzle (Пятнашки) 🧩**: Slide tiles to assemble the pictures of lovely animals.
- **Zoo Sudoku 🔢**: Kid-friendly Sudoku grids in 4x4, 6x6, and 9x9 formats.
- **Zoo Bulls & Cows 🐂**: A classic logic guess-the-sequence game made fun and simple.
- **Zoo Crossword 🧮**: Solve fun, child-friendly word puzzles.
- **Zoo Pyramid 🔺**: Match tiles to clear the pyramid board.

## ✨ Key Features
- **Bright & Cheerful Design**: Custom "Candy" color palette, custom typography (Fredoka font), and smooth micro-animations.
- **Confetti & Particle Effects**: Celebrating every win with colorful animations.
- **Sound Effects 🔊**: Delightful sound cues for moves, combinations, wins, and losses.
- **AI Auto-Solver Helper 🤖**: Stuck? Let the built-in AI solve the game for you or show you the next best moves.
- **Responsive Layout**: Works beautifully on mobile, tablet, and desktop screens.
- **No Complex Setup**: Pure Vanilla HTML, CSS, and JS. No bundlers or heavy dependencies required.

## 🚀 How to Run
Simply open the `index.html` file in any modern web browser! No installation or build steps are necessary.

Alternatively, you can run a local server if you want to test sound files locally:
```bash
npx serve .
```

## 🎮 Игры на платформе
- **Zoo 2048 🐾**: Сдвигайте плитки с милыми животными, объединяйте их и доберитесь до волшебного Единорога!
- **Zoo Пятнашки 🧩**: Передвигайте плитки, чтобы собрать изображение животного.
- **Zoo Судоку 🔢**: Детское судоку в размерах 4x4, 6x6 и классическом 9x9.
- **Zoo Быки и Коровы 🐂**: Классическая игра на угадывание секретного кода из животных.
- **Zoo Кроссворд 🧮**: Забавные кроссворды на тему животных для развития словарного запаса.
- **Zoo Пирамида 🔺**: Увлекательная математическая пирамида для юных математиков.

## ✨ Особенности
- **Премиум-дизайн**: Яркая тема "Candy", красивый шрифт Fredoka и плавная анимация.
- **Праздничные эффекты 🎉**: Анимация конфетти при победе.
- **Звуки 🔊**: Звуковые эффекты для действий, побед и поражений.
- **ИИ-Помощник (Авторешение) 🤖**: Встроенный робот-помощник покажет правильные ходы или решит головоломку за вас.
